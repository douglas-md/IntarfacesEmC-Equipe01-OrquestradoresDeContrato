# Fase 07 — Repository JSON

## Persistência de Eventos Acadêmicos usando arquivo JSON

- Arquivo gerado: `eventos_academicos.json`
- Serialização em camelCase, ignorando campos nulos, formato indentado.
- Métodos implementados: Add, GetById, ListAll, Update, Remove.
- Para testar, execute o projeto e confira que o arquivo `eventos_academicos.json` será criado na pasta de execução com os dados registrados.
- Para esta fase, troque a implementação do repositório para `JsonEventoRepository` (confira em Program.cs).
